Три новые фигуры: 

Jester (Шут): Ходит как конь, но только "поедает" фигуры. Просто передвигаться на пустую клетку он не может.

Tank (Танк): Ходит только на 1 клетку во все стороны (как король), но не может быть съеден пешками.

Ghost (Призрак): Может перепрыгивать через любые фигуры (как конь), но ходит только по прямой на 2 или 3 клетки.
